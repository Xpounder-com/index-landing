# Index Landing

Static IDX landing page and wireframe prototypes.

## Local Development

```sh
npm install
npm run dev
```

Open the printed local URL. The main landing page is served from `/`.

Useful project pages:

- `/` - current landing page
- `/IDX Landing.html` - source landing page snapshot
- `/IDX Landing v1.html` - earlier landing page snapshot
- `/IDX Wireframes.html` - interactive wireframe canvas
- `/IDX Wireframes v2.html` - generated wireframe bundle
- `/IDX Wireframes v2-print.html` - print layout

## Build And Smoke Test

```sh
npm run build
npm run smoke
```

The build output is written to `dist/`.
